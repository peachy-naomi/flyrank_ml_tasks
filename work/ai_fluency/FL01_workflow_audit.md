# AI Fluency FL-01: Workflow Audit

## 1. Task Classification Table
| Task | Classification | Rationale |
| :--- | :--- | :--- |
| **1. Debugging Python syntax errors** | Delegate to AI | AI is much faster at spotting missing commas or indentation errors than a human. |
| **2. Learning core ML theory** | **Just Me** | I need to internalize these concepts for interviews; AI shortcuts prevent deep learning. |
| **3. Brainstorming Research Questions** | Collaborate with AI | AI helps expand my initial ideas, but I must choose the one that fits the data. |
| **4. Writing Internship Weekly Reports** | Delegate to AI | I provide the bullet points of my work; AI handles professional formatting and tone. |
| **5. Explaining complex code logic** | Collaborate with AI | I explain my understanding, and AI confirms or flags potential logic gaps. |
| **6. Peer-reviewing teammates** | **Just Me** | Feedback requires human empathy and personal accountability which AI cannot provide. |
| **7. Formatting Bibliography/Citations** | Fully Automate | This is a low-value, high-precision task that tools handle perfectly without human input. |
| **8. Summarizing documentation** | Delegate to AI | High-volume reading is perfect for AI to condense into actionable bullet points. |
| **9. Feature Engineering ideas** | Collaborate with AI | AI suggests new data signals; I verify if they are "leaky" or logically sound. |
| **10. Practicing Coding Challenges** | **Just Me** | Using AI for practice defeats the purpose of building technical muscle memory. |

## 3. Target Tasks & Success Definitions
These three tasks will be reused in assignments FL-02 through FL-04.

### Task 1: Debugging ML Pipeline Errors
*   **Done Well Means:** The error is resolved, and I can explain the root cause and the fix in one sentence without further AI help.

### Task 2: Technical Summary Writing
*   **Done Well Means:** A complex notebook or report is condensed into exactly 3 bullet points that a non-technical manager can understand.

### Task 3: Feature Discovery
*   **Done Well Means:** I use AI to brainstorm 5 new features, and at least one of them leads to a measurable increase in my model's Precision@50 score.
